# 文件目录

> 用于查询共享目录和文件信息

## 获取数据

```
GET /share
```

> 请求参数

<table>
    <tr>
        <th style="width:150px;">属性名称</th>
        <th style="width:60px;">类型</th>
        <th style="width:60px;">必填</th>
        <th style="width:200px;">说明</th>
    </tr>
    <tr>
        <td>dir</td>
        <td>string</td>
        <td>-</td>
        <td>目录, 默认:/, /代表共享目录根目录</td>
    </tr>
</table>

> 返回指定dir目录下的目录和文件列表
