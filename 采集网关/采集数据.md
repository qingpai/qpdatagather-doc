    <tr>
        <th style="width:150px;">名称</th>

        <td>设备ID</td>

## 获取数据

```
GET /api/v1/data
```
> 请求参数

<table>
    <tr>
        <th style="width:150px;">属性名称</th>
        <th style="width:60px;">类型</th>
        <th style="width:60px;">必填</th>
        <th style="width:200px;">说明</th>
    </tr>
    <tr>
        <td>id</td>
        <td>string</td>
        <td>-</td>
        <td>设备ID</td>
    </tr>
    <tr>
        <td>remove</td>
        <td>bool</td>
        <td>-</td>
        <td>获取后是否删除, 默认true, 不删除传false</td>
    </tr>
</table>

## 提交数据

```
POST /api/v1/data/:id
```
